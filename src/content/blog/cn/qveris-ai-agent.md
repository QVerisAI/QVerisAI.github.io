近日，QVeris AI与 网宿科技 达成战略合作。双方将围绕企业级 AI 智能体落地，共同开拓 Agent Infra 市场，并推进面向企业客户的“模型+工具”一体化解决方案，为智能体提供从思考决策到安全执行的全栈支撑。

![](../../../assets/blog-qveris-ai-agent-2.png)

来源：中国证券网

这次合作的核心，是面向企业级 Agent 落地所需的关键基础设施进行协同建设。

过去一年，大模型让企业看到了 AI 在知识理解、内容生成、任务规划等方面的巨大潜力。但在真实业务环境中，企业要让智能体进入生产流程，仅有模型能力还不够。智能体还需要稳定连接真实世界的工具、数据、API 和业务系统，并在可控、安全、可审计的前提下完成执行。

这正是企业级 Agent 从 Demo 走向生产环境时最关键的一层基础设施。

企业 Agent 的关键问题：不是“会不会回答”，而是“能不能安全执行”

今天，很多企业已经开始尝试将大模型接入客服、营销、运维、数据分析、办公协同和业务自动化场景。但真正进入生产环境时，企业通常会遇到几个共性问题：

- 工具和 API 高度分散，每个系统都有不同的接口、参数、鉴权和返回格式。

- Agent 在调用工具前缺少可靠的能力发现、质量评估和成本判断机制。

- 单个工具接入容易，但规模化接入数十、数百甚至上万个工具时，开发和维护成本快速上升。

- 企业需要权限控制、密钥管理、执行审计、成本账单和安全边界，不能让智能体以黑盒方式调用外部系统。

- MCP、插件或函数调用机制解决了“能接入”的问题，但企业还需要更高层的能力路由、质量治理和执行闭环。

  

因此，企业级 Agent 的竞争力，不只取决于模型本身，也取决于它可连接、可选择、可调用、可治理的真实世界能力边界。

QVeris AI 对这个问题的判断是：Agent 时代需要一层新的基础设施，把分散的外部 API、数据服务、企业系统和工具能力，转化为智能体可以发现、理解和安全调用的能力网络。

QVeris AI：Agent 的能力路由与执行基础设施

QVeris AI 当前定位为 AI Agent 的能力路由网络，核心流程可以概括为 Discover、Inspect、Call。

Discover：智能体或开发者可以通过自然语言描述需求，动态发现合适的工具和能力，而不需要提前记住具体 API 名称或逐个查阅文档。

Inspect：在执行调用前，系统可以查看工具参数、调用方式、延迟、成功率、成本等关键信息，帮助 Agent 和开发者做出更可靠的选择。

Call：在确认能力和参数后，通过统一接口完成工具调用，并返回结构化结果，便于 Agent 继续推理、验证和执行后续任务。

**围绕这一流程，QVeris AI 构建了四层关键能力**：

- 聚合接入层：统一注册、统一鉴权、统一计费，降低企业对接外部工具和数据源的成本。

- 质量评估层：通过实时拨测、可靠性指数、延迟和成功率评估，帮助 Agent 避免盲目调用。

- 智能匹配层：基于用户意图、场景需求和能力供给进行动态匹配，提升工具选择准确率。

- 可信执行层：围绕权限、安全、成本、审计和风险控制，保障企业级 Agent 调用外部能力时可控、可追踪、可治理。

  

QVeris AI 希望解决的不是“再接一个工具”，而是让智能体拥有一套可复用、可扩展、可治理的执行基础设施。

“模型+工具”一体化：让智能体进入真实业务流程

企业使用 Agent 的最终目标，不是让系统生成更漂亮的文本，而是让它完成真实任务。

一个完整的企业级 Agent 工作流通常包括目标理解、任务拆解、工具选择、外部系统调用、结果验证和业务动作执行。模型负责理解、规划和决策，工具层负责连接真实世界的数据、系统和操作接口。

如果只有模型，Agent 很容易停留在问答和内容生成层面。如果只有工具，系统又缺少自然语言理解、复杂任务规划和上下文推理能力。只有当模型能力与工具执行层被系统化连接起来，企业 Agent 才有可能真正进入业务流程。

网宿科技与 QVeris AI 的合作，正是围绕这一关键环节展开。双方将结合企业级基础设施、场景承载能力与 Agent 能力路由、工具调用和可信执行能力，帮助企业更快构建可落地、可运行、可治理的智能体系统。

面向企业客户的联合价值

通过本次合作，QVeris AI 将与网宿科技共同面向企业客户探索 Agent Infra 场景，重点帮助客户解决以下问题：

- 降低工具和 API 集成成本：企业无需为每个外部工具和业务系统重复编写适配代码。

- 提升 Agent 执行成功率：通过能力发现、参数检查、质量评估和结构化调用，减少工具调用失败和模型幻觉。

- 缩短 Agent 应用落地周期：让企业从模型试验更快进入业务流程验证和生产部署。

- 增强安全与治理能力：围绕权限、密钥、成本、日志和审计建立企业级执行边界。

- 扩展智能体能力边界：通过统一能力网络，让 Agent 可以连接更多真实世界的数据、服务和业务系统。

    
  

对于正在推进 AI 转型的企业来说，这意味着 Agent 不再只是一个对话入口，而可以成为连接业务系统、外部数据和自动化流程的执行入口。

面向典型场景的联合探索

双方后续将围绕多类企业级场景展开联合方案探索，包括金融投资、舆情监测、企业数据分析与报告生成、经营管理、营销与增长自动化、出海业务支持等方向。

这些场景的共同点是：业务价值不来自单次问答，而来自智能体能否稳定、安全、低成本地调用真实系统并完成任务。

QVeris AI 将持续围绕 Agent 的能力路由、质量评估、统一调用和可信执行能力进行产品建设，与生态伙伴共同推动企业级智能体从概念验证进入生产落地。

创始人观点

**QVeris AI 创始人兼 CEO 王林芳表示**：

“企业级 Agent 的落地，核心不只是模型能力，而是模型能否安全、稳定地连接真实世界的工具和业务系统。QVeris AI 一直专注于构建 Agent 的能力路由与执行基础设施，让智能体能够发现、评估并调用外部能力。与网宿科技的合作，是我们推动企业级 Agent 从‘会思考’走向‘会执行’的重要一步。”
